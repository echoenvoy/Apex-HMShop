
# 🛍️ HMShop – Modern eCommerce Frontend

HMShop is a **responsive eCommerce Single-Page Application (SPA)** built with **HTML, CSS, and Vanilla JavaScript**.
It includes product browsing, search, filtering, cart, wishlist, and checkout simulation — fully on the frontend with `localStorage`.
This project was developed using a **vibe coding** workflow, leveraging AI-assisted development to accelerate implementation.

---

##  Main Features

*  **Shopping Cart** (add, update, remove, saved in localStorage)
*  **Wishlist System**
*  **Search + Filters** (category, price range, stock)
*  **Checkout Simulation** (shipping + payment form)
*  **Light/Dark Mode**
*  **Fully Responsive UI**
*  **Toast notifications & smooth animations**
*  **SPA Navigation** (no page reloads)

---

##  Tech Stack

* **HTML5, CSS3, Vanilla JavaScript**
* **DummyJSON API** for product data
* **localStorage** (cart, wishlist, theme)
* **Font Awesome + Google Fonts**
